This is the UI sample application.
It has been provided to help create, test and view UI custom controls for project Fiori.
This application provides a easy way to add and check new controls that are being developed.

To add a new control:

1) Update util\Id.js with control ID
2) Update util\Title.js with control Name
3) Update util\Navigation.js with control ID and Name
4) Add control test page view and controller
