Visual Business UI5: Rendering multiple routes.

The aim:

To check that Routes and Arrows display correctly.


To test:

  1. Check that the GeoMap has loaded, and check that you can see the following Visual Objects (you may need to zoom out to see some of the Visual Objects):
    1. Check that you can see Routes and Arrows.

    2. Check that you can see a red colored route, and a blue coloured route.

    3. Check that some routes have larger width than others.

    4. Check that some routes are bi-directional, and some point in one direction only.
