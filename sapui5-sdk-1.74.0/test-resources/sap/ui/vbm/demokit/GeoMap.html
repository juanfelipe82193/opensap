VBI GeoMap

Introduction

The Visual Business GeoMap control wraps the VBI control and exposes a simple UI5 interface. For UI5 developers it is easier to use, but it fits only for client side applications.

Examples

Plain Map

This example shows the creation of an plain map focus on a given position. In the example we define only the width and height of the control. If ommited the default size of 800x600 pixels would be chosen. The size can also be given relative.

The map data comes per default from MapQuest and is based on OpenStreetMap data. The used map can be changed by providing a dedicated map configuration to the control.

Objects on the map

This sample shows the control creation with additional object placed on the map. The most common object to use is a Spot based on a given location. The sample shows the template based cration of spots with aggregation binding as well as the exlicite creation of a spots.
