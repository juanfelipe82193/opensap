Visual Business UI5: Loading a VBI Project using Link and Image for Map Provider

The aim:

To check that the different layouts for the Copyright label display correctly.


To test:

Please refresh the page before testing a new size.
  1. Click the "Text" button. Check the Copyright label is located on the bottom-right corner of the map, and that it only has text information.
  2. Click the "Link+Image" button. Check the Copyright label has a link to map source website (https://www.mapquest.com/) and a little image at the end.
  3. Click the "Image" button. Check the Copyright has text but no Link and a little image at the end
  4. Click the "TwoLayerLinkImage" button. Check the Copyright shows two layers, and that one of the layers has a link and an image

    * This test may require you to scroll the map horizontally/vertically to view the Copyright label of the map.
