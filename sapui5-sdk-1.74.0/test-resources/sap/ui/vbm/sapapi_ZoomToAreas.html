ZoomToAreasById

SAP UI5 Visual Business: zoomToAreasById Test

The aim:

To test if we can zoom to one or more areas by id


To test:

  1. Click 'Area One' to zoom to area with id 'AreaOne'
  2. Click 'Area Two' to zoom to area with id 'AreaTwo'
  3. Click 'Area Three' to zoom to area with id 'AreaThree'
  4. Click 'Multiple Areas' to zoom to areas with id 'AreaOne' and 'AreaTwo' and 'AreaThree'
