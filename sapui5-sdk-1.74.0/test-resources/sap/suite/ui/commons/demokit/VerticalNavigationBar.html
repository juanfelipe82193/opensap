VerticalNavigationBar

Introduction

This control extends the sap.ui.ux3.NavigationBar and allows you to display navigation items vertically. The navigation list can contain NavigationItem or CountingNavigationItem controls. The CountingNavigationItem control can display the number of items on a corresponding content area. It also provides a rich tooltip that can appear and disappear after a certain delay.


Keyboard Support

Use arrow keys to navigate through the items and Spacebar or Enter to choose the items.
