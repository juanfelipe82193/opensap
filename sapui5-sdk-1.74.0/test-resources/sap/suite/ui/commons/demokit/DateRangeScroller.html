DateRangeScroller

Introduction

The DateRangeScroller provides a method to scroll through a series of time periods, each of which is represented by a starting date and an ending date, known as the date range. The user may scroll to the previous or next date range. Several predefined ranges are supported such as day, week, month, year, and custom. The user may also set Date Format (to override the control's default formatting) by an API call.

The DateRangeScroller provides screen reader support for users with special accessibility needs. When userâ€™s screen reader is active and user tabs into the DateRangeScroller, the "DateRangeScroller" will be announced and the current date range will be announced.

Keyboard Support

When the input focus is set to the control, it is possible to navigate to previous or next date range using the arrow left / arrow down and arrow right / arrow up keys.
