Inbox

Introduction

Inbox is a work list management tool which combines viewing, searching, filtering and sorting the list of tasks. It also allows performing actions on the list of tasks. Typical task actions are "Claim", "Release" and "Complete". Users can select a Task from the Task List and perform these actions. Multiple execution of tasks is also supported. There are two levels of filtering. In the first level, a pre-defined filter can be selected. The options available are 'My Open Tasks', 'My Completed Tasks', 'My Escalated Tasks' and 'My Overdue Tasks'.
